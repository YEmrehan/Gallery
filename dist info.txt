JAR Dosyasını Çift Tıklayarak Çalıştırmak İçin:
JAR dosyası, hem JDK hem de JRE ile çift tıklanarak çalıştırılabilir.
Ancak çift tıklama işe yaramıyorsa,
büyük ihtimalle NetBeans'de projeyi yukarıdaki yeşil ok simgesiyle
çalıştırmamış olabilirsiniz.
Projeyi çalıştırdığınızda, NetBeans sizden bir Main-Class seçmenizi
isteyecektir. Bu işlemi tamamladıktan sonra, Clean and Build işlemi
yapmanız gerekir. Böylece oluşturulan JAR dosyasını hem JRE hem de JDK
ile çift tıklayarak sorunsuz bir şekilde çalıştırabilirsiniz.

To Run the JAR File by Double-Clicking:
A JAR file can be executed by double-clicking with both JDK and JRE.
However, if double-clicking doesn't work, it's most likely because
you haven't run the project in NetBeans using the green play button
at the top.
When you run the project, NetBeans will ask you to select a Main-Class.
After selecting it, you need to perform a Clean and Build operation.
Once this is done, the generated JAR file can be run by double-clicking
with both JRE and JDK without any issues.
